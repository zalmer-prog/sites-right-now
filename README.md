# Sites Right Now

Marketing site for **Sites Right Now** — fast, affordable websites for small businesses.
One flat plan: **$75/month** covering domain, custom design and hosting. Extra features on request.

## Stack
Plain static site — `index.html` + `styles.css`. No build step.

## Local preview
Open `index.html` in a browser, or serve the folder:

```
python3 -m http.server 8000
```

## Deploy
Static host (Cloudflare Pages). Point the domain (e.g. sitesrightnow.com) at the deployment.

## To customize
- **Name / brand:** search `Sites Right Now` in `index.html` and the `◐` brand mark.
- **Price:** search `75` in `index.html`.
- **Contact:** `hello@sitesrightnow.com`.
- **Portfolio:** the "Work" section — swap in real screenshots as projects grow.
